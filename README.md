# refeed's SwayFX config

SwayFX (git master / 0.5.3) desktop on Fedora 42.

## Components

- **sway** — tiling WM config (`~/.config/sway/config`)
- **waybar** — status bar + modules + scripts
- **rofi** — Wayland app launcher with frosted glass theme
- **mako** — notification daemon

## SwayFX build

Runs from a from-source build at `/opt/swayfx-git` (separate login session "SwayFX (git)").
See memory notes for the update procedure.
